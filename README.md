# ✨ Ritika's Personal Portfolio

A clean, modern, and highly responsive personal portfolio website built with **Next.js**. It is designed to showcase skills, projects, and professional experience with a polished and professional aesthetic. 

The design incorporates tasteful colors, smooth spacing, elegant typography, and subtle animations that bring the interface to life without feeling overwhelming.

## 🚀 Features

- **Hero Section:** A striking introduction with a brief headline and call-to-actions.
- **About Section:** A concise area to share your personal story and professional background.
- **Skills Display:** Visually appealing presentation of technical and soft skills.
- **Projects Gallery:** Showcase your best work using beautifully designed project cards.
- **Contact Form:** A functional and simple contact section including email and social links.
- **Fully Responsive:** Carefully crafted to look perfect on mobile phones, tablets, and desktop displays.
- **Dark/Light Mode:** Seamless theme switching for better user experience.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **3D Elements:** [Three.js](https://threejs.org/)

## 💻 Getting Started Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <your-repo-url>
   cd <your-repo-directory>
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to [http://localhost:3000](http://localhost:3000) to see the magic happen! ✨

## 📝 How to Customize

This portfolio is designed to be easily updatable. You don't need to dig deep into complex code to change your details.

- **Projects Data:** To add, remove, or modify your projects, simply edit the `src/data/projectsData.ts` file.
- **Personal Info & Links:** Most of the textual content, social media links, and email addresses can be found directly within the page components in `src/app` or specific sections in `src/components`.
- **Styling:** The color palette and global styles can be adjusted in the Tailwind configuration and your main `.css` files.

*Note: Look out for placeholder text and replace them with your actual information.*

## 🌐 Deployment

This project is fully optimized and ready to be deployed on Vercel. 

1. Push your customized code to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and sign in.
3. Click on **Add New...** > **Project**.
4. Import your GitHub repository.
5. Vercel will automatically detect the Next.js framework. Simply click **Deploy**.

Within minutes, your new professional portfolio will be live for the world to see!
